#我十一计划从北京自驾到大兴安岭
我预计的行程是：
day1：北京 -> 国家一号风景大道（罕塞坝，御道口）-> 锡林浩特大草原
day2：锡林浩特大草原 -> 西乌珠穆沁旗草原99号公路起点 -> 乌尔盖湖景区 -> 阿尔山国家森林公园附近
day3:（今天打算在阿尔山玩一天）乌苏浪子湖看晨雾 -> 蓝房子 -> 天池 -> 不冻河 -> 杜鹃湖 -> 大峡谷（这几个景点可以你帮我规划一下顺序）
day4: 阿尔山国家森林公园 -> 成吉思汗拴马桩 -> 呼伦湖景区 -> 满洲里市

day6: 黑山头 -> 卡线（904县道）-> 恩和村 -> 海拉尔
day7: 海拉尔 -> 兴安盟（700km）
day8：兴安盟 -> 北京(1000km)
